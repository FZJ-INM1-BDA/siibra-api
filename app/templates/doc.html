EBRAINS atlascore provides a central REST-API for core functionalities of the EBRAINS human brain atlases, serving both web frontends and python clients.

Referencespaces

The atlascore can provide either a list of all supported referencespaces, or one single referencespace. To get one single referencespace the KnowledgeGraph ID is needed. In Both cases the referencespaces has following format:

                                            {
                                                "cite": <String>,

Datasets

It is possible to request different kinds of datasets, with different possibilities to filter the results.

  1. Request all datasets. When no filter is provided all datasets are returned. It is possible to filter by:
    • referencespace id
    • parcellation id
    • region id
    All filters are combined by and.
  2. Request all datasets for parcellations. It is possible to filter the datatsets by providing a parcellation id or to get all datasets without a filter.
  3. Request all datasets for referencespaces. It is possible to filter the datatsets by providing a referencespace id or to get all datasets without a filter.
  4. Request all datasets for region. It is possible to filter the datatsets by providing a region id or to get all datasets without a filter. Another way to filter the data is to provide a list of region ids.

The provided ids can be either long or short formatted:
